# gRPC Kit Doc

主要涉及HTTP2、gRPC、微服务治理等相关技术内容

## Prerequisites

```
git clone https://github.com/grpc-kit/doc.git
```

clone子模块

```
git submodule init
git submodule update
```

或者一步到位

```
git clone --recursive https://github.com/grpc-kit/doc.git
```

下载依赖的node包

```
npm install
```

## Running the website locally

```
hugo server
```
